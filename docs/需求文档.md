# 需求文档

## 一、专家库

### 1. 专家信息管理
- 支持专家信息的新增、修改、删除等操作。
- 专家信息字段需覆盖：
  - 基本信息：姓名、性别、联系方式、工作单位等。
  - 专业信息：工程类（下设子项）、服务类（下设子项）、货物类（下设子项）。
  - 回避信息：回避单位、回避人员等。
- 单位、职称支持枚举项管理。

### 2. 批量导入与导出
- 支持专家信息批量导入与导出。
- 导入格式：Excel。

### 3. 抽取申请
- 业务操作员可发起专家抽取申请并填写申请信息。
- 抽取需求信息包含：专家专业类别、所需专家人数、抽取方式、评审时间、评审地点等。

### 4. 抽取申请记录
- 支持抽取申请记录的查询与查看。

## 二、抽取规则配置

### 1. 自动匹配规则
- 支持依据项目需求自动匹配抽取规则。
- 覆盖：专家专业匹配、职称要求匹配、回避规则匹配等。

### 2. 自定义抽取方式
- 支持自定义抽取方式，例如随机抽取、摇号抽取等。

### 3. 替补专家
- 支持设置替补专家数量。
- 当主抽专家无法参与评审时，系统自动从替补专家中补选。

## 三、专家抽取执行

### 1. 自动筛选与抽取
- 系统依据抽取需求与规则，从专家库中自动筛选符合条件的专家，并开展随机抽取。

### 2. 抽取结果清单
- 抽取完成后生成结果清单。
- 清单包含：专家姓名、单位、专业、联系方式等信息。
